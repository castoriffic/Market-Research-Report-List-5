<p><ul><li>Horizontal Branch Pipe</li><li>Vertical Branch Pipe</li></ul></p>
<p><p>VRF air conditioners are commonly categorized into two main types: Horizontal Branch Pipe and Vertical Branch Pipe systems. Horizontal Branch Pipe systems use a layout where multiple indoor units are connected to a single outdoor unit through a network of horizontal pipelines, providing flexibility in installation. On the other hand, Vertical Branch Pipe systems connect indoor units to a single outdoor unit through vertical piping, making them suitable for high-rise buildings where vertical installation is more convenient. Each type has its own advantages and is chosen based on specific requirements and building layouts.</p></p>
